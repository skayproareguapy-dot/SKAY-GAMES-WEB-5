# SKAY GAMES Web

Proyecto listo para abrir en VS Code, subir a GitHub y desplegar en Vercel.

## Pasos

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Backup

Se incluye el archivo `skaygames-backup-2026-04-20-21-21-17.json` como respaldo exportado desde el panel admin.
